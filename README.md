# Accessible Oman Map 🗺️

## 📱 Project Idea
A mobile application built with Flutter that helps users find wheelchair-accessible places in Oman.

## ❗ Problem
Many people in Oman face difficulty finding accessible and wheelchair-friendly places.

## 🎯 Purpose
This project supports Oman Vision 2040 by improving accessibility and helping people with disabilities locate suitable places easily.

## 🎯 Objective
To create a smart solution that maps accessible locations and improves inclusivity in Oman.

## ⚙️ Features
- User login system (Supabase Authentication)
- Interactive map of Oman
- Add accessible places
- Rate and review locations
- View accessibility details of places

## 🛠️ Technologies Used
- Flutter
- Dart
- Supabase
- Flutter Map
